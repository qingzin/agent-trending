# Contributing

欢迎帮助这个仓库变得更清晰、更准确、更易用。

## 适合提交的内容

- 论文链接错误修复
- 错别字、格式问题修复
- 缺失条目补充
- 目录结构和导航优化
- README 与说明文档改进

## 提交建议

- 尽量保持原有目录结构稳定
- 对新增内容给出清晰来源
- 涉及大段增补时，优先说明整理依据
- 不要直接删除大块内容，除非能说明原因

## 对版权与引用的要求

- 请只提交你有权公开整理的内容
- 论文原始版权归原作者所有
- 如引用外部资料，请尽量附上来源链接

## Pull Request 说明

提交 PR 时建议简要说明：

- 改了什么
- 为什么要改
- 是否影响已有结构或命名

## 交流方式

如果改动较大，建议先开 issue 讨论，再发 PR。
